# Tech Stack: 52 Projects Tracker

## Frontend
- **Vanilla HTML5:** For structure.
- **Vanilla CSS3:** For styling, focusing on a dark mode minimalist aesthetic with custom color-coded project statuses, a 4-column grid layout, and a grid-based stats visualization.
- **Vanilla JavaScript (ES6+):** For client-side logic and interaction.

## Backend
- **Node.js (Express):** To serve the application and provide API endpoints for data management.

## Database
- **SQLite:** For structured data persistence of projects, statuses, and global notes.

## Development Tools
- **Gemini CLI:** For project management and assistance.
